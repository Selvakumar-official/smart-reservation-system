# College Bus Tracking System

## Description
The College Bus Tracking System allows students to track the live location of their college buses in real time. Drivers send GPS coordinates using their mobile device, and students can view the bus location on a map.

## Features
- Live bus tracking using GPS
- Google Maps integration
- Student login system
- Driver GPS location sharing
- Firebase Realtime Database
- Bus routes information
- Simple admin dashboard

## Technologies Used
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Firebase Realtime Database
- Firebase Authentication

APIs:
- Google Maps API
- Geolocation API
